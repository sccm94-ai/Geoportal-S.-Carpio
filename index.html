Objetivo

Presentar un geoportal web funcional que permita visualizar capas geográficas publicadas desde GeoServer, descargar información en varios formatos interoperables y ofrecer una interfaz clara para navegación, consulta y contacto.

Visualización y cambio de imagen WMS

Usa los botones para mostrar cada capa simbolizada y también la composición de las 3 capas.

Selecciona una capa para generar la vista WMS.
Esta sección usa OpenLayers para cambiar la vista WMS. Si las capas aparecen muy a una esquina, solo ajusta el valor de bbox4326.

Descarga de capas desde GeoServer

Cada capa puede descargarse en Shapefile, KML y GML usando servicios WFS del GeoServer.

Shapefile normalmente se descarga como ZIP. KML y GML dependen de los formatos habilitados en WFS.

Visor cartográfico en OpenLayers

Mapa base OpenStreetMap con capas WMS, barra de escala y coordenadas en latitud/longitud.
